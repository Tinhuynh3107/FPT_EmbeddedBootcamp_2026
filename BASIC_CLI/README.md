# OVERVIEW
*| This is a simple program that handles basic commands (CLI) using the C language. The goal of this exercise is to build a basic command-line interface for beginners. It works by implementing a simple command parser that reads user input, interprets commands, and executes corresponding functions in a simulated environment.*

## HOW TO USE
Follow these steps to run the program and make sure you have GCC installed.<br>
   **Clone this repository**
   ```
   git clone https://github.com/Tinhuynh3107/FPT_EmbeddedBootcamp_2026
   ```
**Go into the repository**
   ```
   cd FPT_EmbeddedBootcamp_2026/BASIC_CLI
   ```
**Compile the source code**
   ```
   make all
   ./main.exe
   ```
**Try available commands:**<br>
